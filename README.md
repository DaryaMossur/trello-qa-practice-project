# Trello QA Practice Project

## Overview
This repository contains manual testing documentation created during QA practice testing of a Trello-like web application.

## Scope
The following types of testing were performed:
- Functional Testing
- UI Testing
- Exploratory Testing
- Basic Responsive Testing

## Test Documentation
### Checklist
Contains main verification points for:
- Authentication
- Boards
- Lists
- Cards
- Search
- Responsive UI

### Test Cases
Includes detailed test scenarios with:
- Preconditions
- Steps to reproduce
- Expected results
- Actual results
- Statuses

### Bug Reports
Contains documented defects with:
- Severity
- Priority
- Reproduction steps
- Expected and actual results

- Google Sheets
